# 여우방 팔로우리스트 V8 - 조별복사 전용

## 적용 방법
1. ZIP 압축을 풉니다.
2. GitHub 저장소에서 기존 파일을 삭제하거나 덮어씁니다.
3. 아래 파일을 업로드합니다.
   - index.html
   - style.css
   - script.js
   - manifest.json
   - sw.js
   - icon-192.png
   - icon-512.png
4. Commit changes를 누릅니다.
5. 2~3분 후 사이트를 새로고침합니다.

## 포함 기능
- 구글시트 Sheet2 데이터만 표시
- 샘플 데이터 없음
- 공지 팝업 없음
- 전체복사 없음
- 사람별 복사 없음
- 조별복사만 제공
